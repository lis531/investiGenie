# investiGenie

investiGenie to projekt oparty na Pythonie, stworzony w celu wspierania analizy inwestycyjnej i podejmowania decyzji finansowych. Repozytorium zawiera narzędzia i funkcje ułatwiające przetwarzanie danych finansowych, analizę oraz raportowanie.

## Funkcje

\- Pobieranie i wstępne przetwarzanie danych finansowych  
\- Algorytmy analizy inwestycji  
\- Narzędzia do raportowania i wizualizacji  
\- Rozszerzalna architektura dla własnych strategii

## Instalacja

Sklonuj repozytorium:

`git clone https://github.com/lis531/investiGenie.git`

Zainstaluj zależności:

`pip install -r requirements.txt`

## Użytkowanie
Uruchom główną aplikację:

`python main.py`

Szczegółowe instrukcje znajdziesz w dokumentacji.
